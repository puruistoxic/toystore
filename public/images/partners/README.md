# Partner Logos

This directory should contain the official logos of authorized partners.

## Required Logo Files:

1. **cpplus.png** - CP Plus logo
2. **hikvision.png** - Hikvision logo  
3. **panasonic.png** - Panasonic logo
4. **godrej.png** - Godrej logo
5. **concox.png** - Concox GPS logo

## Logo Specifications:

- **Format**: PNG with transparent background (preferred) or JPG
- **Size**: Recommended 200-400px width
- **Aspect Ratio**: Maintain original brand proportions
- **Quality**: High resolution for crisp display

## How to Add Logos:

1. Download official logos from partner websites or partner portals
2. Ensure you have permission to use the logos
3. Resize if needed while maintaining aspect ratio
4. Save with the exact filenames listed above
5. Place in this directory

## Fallback:

If local logos are not available, the website will attempt to load logos from Clearbit's logo service. If that also fails, it will display the brand name as text.

## Brand Guidelines:

Please ensure all logos are used in accordance with each partner's brand guidelines and usage policies.
