# Hero Carousel Images

This directory contains the hero banner images for the homepage carousel.

## Required Images

Please add the following images to this directory:

1. **hero-main.jpg** - Main hero image for "Professional Security & Tracking Solutions"
   - Recommended size: 1920x800px or larger
   - Format: JPG, PNG, or WebP
   - Should represent your security and tracking services

2. **cctv-systems.jpg** - CCTV surveillance systems image
   - Recommended size: 1920x800px or larger
   - Should showcase CCTV cameras and surveillance equipment

3. **gps-tracking.jpg** - GPS tracking solutions image
   - Recommended size: 1920x800px or larger
   - Should represent vehicle tracking and fleet management

4. **maintenance-services.jpg** - Maintenance and repair services image
   - Recommended size: 1920x800px or larger
   - Should show maintenance technicians or equipment

## Image Guidelines

- **Aspect Ratio**: 2.4:1 (width:height) works best for the carousel
- **File Size**: Optimize images to keep file sizes reasonable (under 500KB each if possible)
- **Format**: JPG for photos, PNG for graphics, WebP for best compression
- **Quality**: Use high-quality images that represent your brand professionally

## Fallback

If images are not found, the carousel will automatically fall back to a gradient background matching your brand colors.
